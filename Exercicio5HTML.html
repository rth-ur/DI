Exercício 5 HTML - Principais Elementos

Os principais elementos html incluem tags para estrutura, texto, mídia e outros tipos de elementos que podem trazer mais interatividade com o documento html. Essas tags são interpretadas pelos navegadores e então renderizadas e mostradas para os usuários.

A seguir vamos descrever as principais tags não estruturantes

Tag p - Parágrafo

a tag parágrafo é utilizada para definir parágrafo. Neste bloco estamos usando-a, esta tag vira dentro do sinal do diamante

Tag br

A tag br é utilizada para quebras de linhas e podemos utiliza-la quantas vezes quisermos. Neste exemplo usaremos três quebras de linhas de espaço para o proximo paragrafo.




Tag hr

A tag hr é aquela que define/cria uma linha horizontal que possue a largura do documento html. Como exemplo vamos criar uma linha que vai separar esse parágrafo do proximo
