O que é o Grid Layout?

O grid Layout é um técnica avançada para a criação de layouts complexos usando linhas e colunas, ou seja ele é bidmensional, diferentemente do flex blox que é unidimnecional baseado apenas em linha.

Flexbox ou GridLayout?

O flex blox e o Grid Layout podem ser utilizados em conjunto e voce pode por exemplo aplicar regras do Flexbox a elementos que estejam usando o layout de grade do css, aporveitando assim o melhor das duas técnicas.

A escolha entre Flexbox e Grid Layout não se trata de um substituir o outro mas sim de entender as necessidades do seu layout e escolher a técnica que melhor resolve o problema elicitado.,
