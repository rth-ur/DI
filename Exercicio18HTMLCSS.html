Exemplo 18 HTML CSS - Vinculação Interna

Incorporada

A vinculação de arquivos CSS de forma interna pode ser realizada de várias formas diferentes. Uma delas é a vinculação incorporada, que é aquela onde a regra CSS é aplicada dentro da página e geralmente vem no cabeçalho da página dentro do atributo style.
