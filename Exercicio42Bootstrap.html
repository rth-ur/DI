O Bootstrap utliza um sistema de cores contextuais para facilitar a marcação e a estilização de elementos com base em seu significado ou propósito. Essas classes de cores são usadas para dar significado visual aos elementos HTML. Geralmente utilizamos essas cores em botões, alertas, textos, barra de navegação que são utilizados em interfaces de usuários.
