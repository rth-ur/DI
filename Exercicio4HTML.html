Exercicio 4 - HTML - Estrutura Geral

Um documento hmtl é composto por ttags estruturantes que permitem ao navegador imterpretar e exibir informaçoes diversas aos seus usuarios

O documento html é composto pelas seguintes tags estruturantes e por uma instrução, que são descritas a seguir:

instrução DOCTYPE html

A instrução doctype html tem a função de dizer para o navegador qual é a versão do html que sera aplicado ao documento criado. Essa instrução deve vir na primeira linha do documento html é importante para a renderização do documento html que será feito pelo navegador. Lembro que o DOCTYPE html é uma instrução e NÂO UMA TAG.

Tag html

A tag html é uma tag estruturante, ou seja, é a tag raiz do documento html, essa tag envolve todo documento estando presente no inicio e no fim do documento. Dentro dela teremos as outras tags estruturanbtes que são head e body, além de diversas outras tags que poderão estar nas tags filhas

Tag head

A tag head é uma tag de cabeçalho os elementos html que estiverem dentro dela não seram visíveis e serão utilizadas para configurações utilizadas pelo navegador para exibir as informações visíveis. Essa tag possui apenas um unico elemento visivel que é o título do documento html, que é representado pela tag title.

A tag head contém tags de metadados sobre a página, codificação, links para css e etc.

Tag body

A tag body inclui todos os elemntos que ficarão visíveis no documento html, tais como textos, mídias, arquivos de audios, imagens e etc. Em síntese, tudo que estiver dentro do body ficará visível para o usuário, salvo em algumas exeções
