Exemplo 30 CSS - FlexBox

O Flexbox é um modelo de layout em CSS projetado para facilitar a criação de layouts complexos e responsivos.

Principais propriedades do Flexbox

Flex-Direction

A propriedade flex-direction define a direção na qual os elementos HTML serão exibidos.
