Exemplo 41 - Bootstrap

Bootstrap

O Bootstrap é um framework CSS muito popular no mundo do desenvolvimento de sistemas. Esse framework foi criado em 2011 pelo Twiter. Ele possibilita estilizar elementos HTML de forma fácil e intuitiva.

Seletores de Classe Container e Container-fluid

Containers de uma forma geral são caixas que contém outros elementos dentro dela. Essas caixas tem como objetivo, organizar, alinhar e posicionar elementos em uma tela. No contexto do Bootstrap nós temos dois tipos: container e container-fluid

No caso do seletor container, tudo o container terá larguras fixas respeitando as margens a direita e à esquerda. Neste caso o conteúdo ficará centralizado no centro da tela. Neste exemplo estamos usando o seletor de classe do Bootstrap container, além de usar também um alert na cor azul claro.

Container-fluid

O container-fluid, diferentemente do seletor container, ocupa 100% da largura de uma tela. Ele é muito utilizado quando queremos que o conteúdo ocupe toda a tela disponível, por exemplo que quando queremos constrir templates administrativos, dashboard, etc.

Neste exemplo, estamos usando o container-fluid compare com o exemplo acima que usa apenas o seletor container.
