Exemplo Método Map

Seja o vetor de números pares de 0 ate 10 [0,2,4,6,8,10]

Vamos criar um novo vetor clicando no botão usando o método Map para multiplicar os elementos por 5 e exibir o novo valor.

Para fazer isso poderíamos usar a estrutura: 

                    for (let i = 0; i < numeros.length; i++) {
                        numeros[i] = numeros[i] * 5;
                    }

Ou usando Map: 

                    const numsMultiplicados = numeros.map(function (numero) {
                        return numero * 5;
                    });

                    ou usando Arrow function
                    const numsMultiplicados = numeros.map(numero => numero * 5);
