Exemplo 7 html - Elementos de Lista (ul e ol)

Os principais elementos htmls para listas não ordenadas e ol para listas oredenadas. Esses elementos são muito utilizadas para exibir listas e de forma muito clara são utilizados também para a criação de menus. A seguir descrevemos cada um deles.

Lista Não Ordenada ul

A lsita não ordenada é utilizada para agrupar um conjuntos de itens onde a ordem não é relevante. Como por exemplo uma lista de livros ou de compras. Os elementos dessa lista são bullets por padrão.

Para representar uma lista usaremos a tag ul ou ol além da tag item que é representada por li. Abaixo exemplificamos o uso da lista não ordenada -ul. Exemplo

Etapas para desenvolver um Site - Lista Ordenada com subitens

  1. Planejamento
    1. Definir o objetivo do site
    2. Indentificar o publico alvo
    3. Criar um mapa do site (site map)
  2. Design
    1. Criar Wireframes e protótipos
    2. definir a paleta de cores e a tipografia
    3. Desenvolver o layout usando Figma e ou AdobeXd
  3. Desenvolvimento
    1. Configurar um ambiente de desenvolvimento
    2. Codificar o front-end(html, css, js)
    3. Implementar o back-end
      1. Linguagem de Programação OO (C#, Java, Go, Php, Python)
      2. Banco de Dados - SQL Server ou Oracle
      3. Git - Git Hub
  4. Testes
    1. Testar a responsividade em diferentes dispositivos
    2. Realizar testes de usabilidade
    3. Corrigir bugs identificados
  5. Publicação
    1. Escolher um provedor de hospedagem
    2. Configurar o dominio SSl
    3. Fazer o deploy do site.
